centos7.2使用rpm安装jdk8

①下载jdk

去jdk下载页面找到要下载的jdk，用wget下载

②安装jdk

rpm -ivh jdk-8u131-linux-x64.rpm

③配置环境变量(告诉电脑jdk在哪里)

vim /etc/profile //打开环境变量文件

进入insert模式(按字母i) 在最后面加上下面这段，然后按esc 然后保存退出(按 :wq回车)

source /etc/profile //重新载入

④检查是否完成安装

java -version

rmp卸载jdk

rpm -qa|grep jdk   //查看版本
rpm -e  jdk版本 //卸载
